Place custom Cadova promo audio files here if you want to replace the generated Web Audio soundtrack.

Suggested filenames:

- ambient-pad.mp3
- focus-pulse.mp3
- soft-whoosh.mp3
- ui-tick.mp3
- transition-rise.mp3
- outro-chime.mp3

Then update `PROMO_AUDIO_ASSETS` in `src/app/components/promo/usePromoAudio.ts`.
